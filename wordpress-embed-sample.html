Grafana Dashboard Embed Test

Instructions for WordPress:

  1. Copy the iframe code above
  2. In WordPress, add a "Custom HTML" block
  3. Paste the code
  4. Replace the dashboard ID in the URL with your actual dashboard

To get your dashboard URL:

  1. Go to http://localhost:3000
  2. Open your dashboard
  3. Click "Share" → "Link" → "Public Dashboard"
  4. Copy the public URL and use it in the iframe src
